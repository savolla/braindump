Use this *coloring* convention when annotating [[id:ee3b9868-c753-4b24-a2c4-44111a313b09][Book]]s

| Color   | Meaning  |
|---------+----------|
| red     | claims   |
| yellow  | concepts |
| blue    | tips     |
| green   | howtos   |
|---------+----------|
| sepia   | story    |
| orange  | chart    |
| grey    | people   |
| magenta | event    |
