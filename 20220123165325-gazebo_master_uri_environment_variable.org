#+title: GAZEBO_MASTER_URI environment variable
#+filetags: :ros:what_is:

IP and port number of *gzserver*. Clients can also connect to remote servers and run simulations from there. Hence *gzclient* and *gzserver* are not always need to run on the same machine.
